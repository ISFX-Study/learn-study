## :triangular_flag_on_post: 스터디 진행 방향
+ 2주마다 오프라인으로 만나서 각자 발표(10~20분)

+ 발표 자료의 형식은 자유입니다.
+ 세부 일정 및 진행 방향은 추후 변경 될 수도 있습니다.
